# Project - Indian-Food Analysis Power BI 

Project - Big Data - Indian Food - Analysis Dashboard & Reports.
Tools: Power BI
- Analyzed 5,000+ Indian food recipes dataset to uncover trends in cuisines, diet types, and cooking times.
- Built interactive dashboards with bar, pie, and column charts to display cuisine breakdown, dietary distribution, and preparation time insights.
- Helped identify most popular cuisines and recipe categories for strategic decisions.
